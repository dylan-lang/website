
******************
Dylan Package Docs
******************

Documentation for packages in the Dylan package catalog.

.. note:: The documentation for some packages is still part of the Open Dylan
   project, available in the `Library Reference
   <https://opendylan.org/documentation/library-reference>`_.  Over time more
   of those docs will be hosted here as they're moved into their own
   repositories.

* :doc:`base64 <base64/index>` - Base-64 encoding/decoding
* :doc:`binary-data <binary-data/index>` - A DSL for parsing and assembling binary data
* :doc:`command-interface <command-interface/index>` - Interactive command interface system for Dylan
* :doc:`command-line-parser <command-line-parser/index>` - Parse command line flags and subcommands
* :doc:`concurrency <concurrency/index>` - Concurrency utilities
* :doc:`data-analysis <data-analysis/index>` - A collection of data analysis libraries
* :doc:`deft <deft/index>` - Manage Dylan workspaces, packages, and registries
* :doc:`dylan-tool <dylan-tool/index>` - DEPRECATED - use deft instead
* :doc:`hash-algorithms <hash-algorithms/index>` - SHA-1 and SHA-2 hashing
* :doc:`http <http/index>` - HTTP client, server, and templating engine
* :doc:`logging <logging/index>` - Logging library
* :doc:`lsp-dylan <lsp-dylan/index>` - Language Server Protocol
* :doc:`melange <melange/index>` - Generates C-FFI bindings by parsing C header files
* :doc:`meta <meta/index>` - Recursive descent parser DSL
* :doc:`objc-dylan <objc-dylan/index>` - An Objective C / Dylan bridge
* :doc:`regular-expressions <regular-expressions/index>` - Regular expression matching
* :doc:`sphinx-extensions <sphinx-extensions/index>` - Dylan extensions to the Sphinx documentation tools
* :doc:`strings <strings/index>` - String manipulation functions
* :doc:`testworks <testworks/index>` - Unit testing framework
* :doc:`time <time/index>` - A time library
* :doc:`tracing <tracing/index>` - A tracing library based on the Dapper work by Google.
* :doc:`uuid <uuid/index>` - Universally unique identifiers
* :doc:`v3d <v3d/index>` - Small 3d vector library
